 Phase 2 变更摘要 — M2 ML 模式识别

  evaluate.py 新增 4 个函数（~200 行）

  classify_engagement_type(video) — 多维规则引擎，将作品分为 6 类：

  ┌────────┬────────────────────────────────┬──────────────────┐
  │  类型  │            触发条件            │ 华为Nova案例得分 │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 爆发型 │ 高播放 + 高分享率              │ 5.5              │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 参考型 │ 极高收藏率 + 高收藏/点赞比     │ 2.5              │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 口碑型 │ 极高点赞率 + 高评论率 + 低播放 │ 0.0              │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 长尾型 │ 低播放 + 健康互动率            │ 2.5              │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 异常型 │ 指标偏离自然分布               │ 0.0              │
  ├────────┼────────────────────────────────┼──────────────────┤
  │ 普通型 │ 所有指标接近均值（兜底）       │ -                │
  └────────┴────────────────────────────────┴──────────────────┘

  每条分类附带置信度和判定信号列表。

  predict_quality_tier(video) — 0-10 连续质量分

  从 6 个维度累加：
  - 平台质量分（基础 0-10）
  - 团队规模加成（5/10/20人阈值）
  - 角色覆盖度（9 个核心角色检测）
  - 导演声誉（推荐率、人气值 log 映射）
  - 奖项加成（金狮/金雀关键词检测）
  - 社会验证（播放量 log 映射）

  附带置信度（high/medium/low）和驱动因素列表。

  detect_anomaly(video) — 7 维异常检测

  ┌────────────────────────┬───────┐
  │          信号          │ 触分  │
  ├────────────────────────┼───────┤
  │ 收藏率>5%但零评论      │ +0.40 │
  ├────────────────────────┼───────┤
  │ 高播放低质量分(≤2)     │ +0.30 │
  ├────────────────────────┼───────┤
  │ 收藏率远超点赞率       │ +0.25 │
  ├────────────────────────┼───────┤
  │ 点赞率>30%偏离自然分布 │ +0.30 │
  ├────────────────────────┼───────┤
  │ 播放量与粉丝数不匹配   │ +0.25 │
  ├────────────────────────┼───────┤
  │ 分享率极端异常(>15%)   │ +0.25 │
  ├────────────────────────┼───────┤
  │ 高播放但零互动         │ +0.20 │
  └────────────────────────┴───────┘

  综合分 ≥0.40 判为可疑，≥0.15 判为需关注。

  run_m2_analysis(video) — 一键执行全部 3 个子模块。

  app.py API 集成

  - /api/evaluate 流程现在：L1 评分 → M2 分析 → L2 元数据提取 → ECD 审计
  - M2 互动分类结果注入到 l2_metadata，覆盖 LLM/规则推断
  - 响应新增 m2_results 字段

  templates/index.html 前端

  L2 卡片底部新增：
  - 质量预测条 — 渐变色进度条（绿≥7.5 / 黄≥5 / 红<5），显示分数和置信度
  - 异常检测标志 — 绿色"数据正常" / 黄色"需关注" / 红色"可疑"+ 详情
  - 互动信号文本 — 显示分类判定依据（如"播放128K(高传播) · 分享率1.2%(较强)"）
