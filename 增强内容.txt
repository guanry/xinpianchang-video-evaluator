  Phase 3 变更摘要 — Embedding 语义理解 + 相似推荐

  新文件：embedding.py (~140 行)

  EmbeddingIndex 类 — 轻量级语义向量索引，零额外依赖：

  ┌───────────────────────────┬─────────────────────────┐
  │           方法            │          功能           │
  ├───────────────────────────┼─────────────────────────┤
  │ add(video)                │ 索引单条视频            │
  ├───────────────────────────┼─────────────────────────┤
  │ add_batch(videos)         │ 批量索引                │
  ├───────────────────────────┼─────────────────────────┤
  │ find_similar(id, top_k=5) │ 查找最相似的 top_k 作品 │
  ├───────────────────────────┼─────────────────────────┤
  │ save() / _load()          │ JSON 文件持久化         │
  └───────────────────────────┴─────────────────────────┘

  向量构建 — 稀疏特征字典（字符 bigram + 元数据加权）：

  ┌─────────────┬──────┬─────────────────────────────┐
  │    特征     │ 权重 │            说明             │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 标题 bigram │ ×1   │ 字符级 2-gram，天然支持中文 │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 内容 bigram │ ×1   │ 前 300 字符补充语义         │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 标签        │ ×2.5 │ 风格信号                    │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 主分类      │ ×3   │ 品类一致性                  │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 子分类      │ ×3.5 │ 最强品类信号                │
  ├─────────────┼──────┼─────────────────────────────┤
  │ 作者名      │ ×1   │ 用于排除同作者              │
  └─────────────┴──────┴─────────────────────────────┘

  余弦相似度 — 稀疏向量点积 / L2 范数，自动排除同作者、过滤低相似度(<0.05)。

  全局单例 get_index() → 缓存文件 instance/embedding_cache.json

  app.py 集成

  - 搜索端点 — 每次搜索自动将结果索引到 embedding 库
  - 评估端点 — 返回 similar_works 字段（top 5 相似作品），附带 id/title/cover/similarity
  - 索引持久化到磁盘，重启不会丢失
