## 对新片场 API 的建议

要实现第一层评价，API 需要补充以下字段：

```
当前缺失              建议增加
─────────────────────────────────────
resolution_type       video_width, video_height（精确像素）
无帧率                frame_rate（23.976/24/25/30/...）
无编码                video_codec（H.264/H.265/ProRes）
无色彩                color_depth, color_space
无音频                audio_codec, audio_channels, audio_sample_rate
无文件大小            file_size
无拍摄设备            camera_model, lens_model（创作者填写或EXIF读取）
```

这些字段在视频上传时就能获取（FFprobe 秒级完成），零成本，补齐后第一层评价即可完整运作。

## 两个 API 的分工总结

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  单作品 API (/article/{id})                      │
│  ─────────────────────────                      │
│  提供：深度数据（标签/团队/Ranks/quality）         │
│  用于：单作品详情页的全部数据                      │
│  缺失：无对标池、无分辨率/帧率等基础技术元数据      │
│                                                 │
│  搜索 API (/search?cate_id=)                     │
│  ─────────────────────                          │
│  提供：品类作品池 + 创作者聚合 + 地理位置          │
│  用于：百分位对标、品类基准值计算、跨作品对比        │
│  缺失：无标签、无团队详细信息、无榜单数据           │
│                                                 │
│  评价系统需要两者结合：                            │
│    单作品 API → 深度标签 + 团队数据               │
│    搜索 API   → 百分位排名 + 品类基准              │
│    视频文件   → AI 分析画面/声音（缺失的关键一块）  │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 关键洞察

### 这个 PDF 验证了我们三层架构的可行性

PDF (C-Eye 商业初筛报告) 本质上就是我们所定义的**第二层 + 第三层**分析：

| 我们的框架 | PDF 对应内容 | 匹配度 |
|-----------|------------|--------|
| B1 构图 | 逐帧标注景别/构图方式 | 95% |
| B2 运镜 | POV/Match Cut/Dutch Angle/Push In | 90% |
| B3 调色 | 色彩策略完整描述 | 85%（缺色板量化） |
| B4 声音 | 声音层次 + 音画一体分析 | 80% |
| C1 节奏 | 5分段结构 + 碎切节奏分析 | 75% |
| C2 叙事 | 分段叙事功能标注 | 90% |
| C3 情感 | 魔幻现实主义情绪分析 | 70% |
| 商业分析 | 资产平移 + 低成本方案 + 提案话术 | **超出预期** |

**结论：市场已经有人在用 AI 做这件事（C-Eye 剪辑线分析矩阵），说明需求真实存在。新片场如果把这个能力内建到平台里，就是差异化壁垒。**

### PDF 比我们的框架多做了什么

1. **预算估级** — "A级百万级大制作"，这对广告商选团队极其重要
2. **低成本复刻方案** — 不是只说"这片子好"，而是说"你预算不够怎么学"
3. **提案话术** — 直接给广告公司/品牌方 copy-paste 的措辞
4. **资产平移价值** — 这个风格可以卖给哪些品类

这 4 个视角都应该纳入我们的第三层分析。
